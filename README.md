## 🛋️ 스터디 진행 방식

같이 책을 읽는 그룹끼리 자유롭게 정할 수 있어요.

### 내용 정리하기

[`notes/book`](https://github.com/bandibook/bandibook/tree/main/notes/book) 폴더 아래 `책 제목/이름`으로 정리해요.


### Pull Request

- `제목`, `assginee`, `label`을 꼭 추가해주세요.

```
# PR 제목
[책 제목] (-자유롭게-)

# assignee
본인

# label
책 제목과 동일
```
